# Public Assets

## Required File

Place the `aiva3d.png` image file in this directory (`/public/aiva3d.png`).

This image is the mascot character that appears on the right side of the spectator view with 3D floating animations.

### What to do:
1. Save the aiva3d.png image file (the purple/pink astronaut character) to this `/public` folder
2. The file should be named exactly: `aiva3d.png`
3. Once placed, the animated mascot will appear on the spectator view pages

The mascot features:
- 3D rotation and floating effects
- Gentle bouncing animation
- Purple/pink glow effects
- Positioned on the right side of the screen
